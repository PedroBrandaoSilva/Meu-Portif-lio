Cronômetro(GitHub)

Projeto desenvolvido, para apresentar a contagem das horas, minutos e segundos. Ele possui três botões principais: Start, que começa a contagem; Pause, que interrompe temporariamente o cronômetro; e Clear, que reseta o tempo para zero. O projeto foi focado em praticar manipulação de eventos, controle de estado e atualização dinâmica da interface.

Dev Sorteador(GitHub)

Sorteador de números que gera um valor aleatório entre dois números definidos pelo usuário: um menor e um maior. O usuário insere os limites e, ao clicar no botão de sortear, o sistema gera instantaneamente um número dentro do intervalo escolhido. O projeto foi pensado para praticar lógica de programação, manipulação de inputs e geração de números aleatórios, e usada uma lógica para evitar possíveis falhas na aplicação .

Demonstração de Layouts Básicos para Websites(GitHub)

Desenvolvi três layouts de sites simples e responsivos com o objetivo de praticar a criação de estruturas limpas, funcionais e adaptáveis para diferentes tamanhos de tela. Cada layout foi projetado pensando na experiência do usuário, priorizando uma navegação intuitiva, boa hierarquia visual e design minimalista. Neste projeto, trabalhei com conceitos de HTML5 e CSS3, aplicando técnicas de responsividade como media queries para garantir que os sites se comportem bem em dispositivos móveis, tablets e desktops. Esse exercício foi fundamental para reforçar boas práticas de estruturação de páginas e adaptação de layouts para diferentes contextos de uso.
